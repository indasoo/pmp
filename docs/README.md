- **十大知识领域**
- **五大过程组**
  - 启动过程组
    - 仅整合 / 相关方涉及，分别是**制定**项目章程、**识别**相关方。
  - 规划过程组
    - 除整合外，其他知识领域**全都有规划**管理
  - 监控过程组
    - 监控（项目工作 / 相关方参与）
    - 确认（范围）
    - 控制（变更 / 范围 / 进度 / 成本 / 质量 / 资源 / 采购）
    - 监督（沟通 / 风险）
  - 收尾过程组
    - 仅整合涉及，即**结束**项目或阶段
