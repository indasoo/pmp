  • 十大知识领域
    • 三过程类
      • 【质量】规管控
      • 【沟通】规管监
      • 【采购】规施控
    • 四过程类
      • 【相关方】识规管控
  • 五大过程组
    • 启动过程组
      • 仅整合 / 相关方涉及，分别是制定项目章程、识别相关方。
    • 规划过程组
      • 除整合外，其他知识领域全都有规划管理
    • 监控过程组
      • 监控（项目工作 / 相关方参与）
      • 确认（范围）
      • 控制（变更 / 范围 / 进度 / 成本 / 质量 / 资源 / 采购）
      • 监督（沟通 / 风险）
    • 收尾过程组
      • 仅整合涉及，即结束项目或阶段
